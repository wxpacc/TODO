# TODO应用

一个功能完整、现代化的TODO管理Web应用，使用纯前端技术栈开发，支持PWA离线访问。

## ✨ 功能特性

### 📋 基础功能
- ✅ 创建、查看、编辑、删除任务
- ✅ 标记任务完成状态
- ✅ 任务描述和详细信息
- ✅ 任务分类管理（可自定义分类和颜色）
- ✅ 标签系统（支持多标签）
- ✅ 优先级设置（高/中/低）
- ✅ 截止日期设置

### 🎯 高级功能
- ✅ **子任务和检查清单** - 任务分解，进度追踪
- ✅ **任务模板** - 6个预设模板，快速创建任务
- ✅ **重复任务** - 每天/每周/每月/每年自动重复
- ✅ **提醒通知** - 浏览器原生通知，多种提醒时间
- ✅ **多视图切换** - 列表视图 + 看板视图
- ✅ **拖拽排序** - 任务拖拽重新排序

### 🔍 搜索过滤
- ✅ 关键词实时搜索
- ✅ 按分类、优先级、状态过滤
- ✅ 多种排序方式（创建时间、截止日期、优先级）
- ✅ 组合过滤

### 🎨 用户体验
- ✅ 亮色/暗色主题切换
- ✅ 响应式设计（支持手机、平板、桌面）
- ✅ 数据导入/导出（JSON格式）
- ✅ 流畅的动画效果
- ✅ 键盘快捷键支持
- ✅ PWA支持（可离线使用、添加到主屏幕）

## 🚀 快速开始

### 方法一：直接打开
双击 `index.html` 文件，在浏览器中打开即可使用。

### 方法二：本地服务器（推荐）
```bash
# 使用 Python
cd todo-app
python -m http.server 8080

# 或使用 Node.js
npx http-server -p 8080
```

然后在浏览器中访问 `http://localhost:8080`

### 方法三：PWA安装
1. 在浏览器中打开应用
2. 点击地址栏右侧的"安装"按钮
3. 添加到主屏幕，即可像原生应用一样使用

## 📖 使用指南

### 添加任务
1. 在顶部表单中输入任务标题（必填）
2. 可选：添加描述、选择分类、设置优先级
3. 可选：设置截止日期、重复规则、提醒时间
4. 可选：添加标签（用逗号分隔）
5. 点击"添加任务"按钮

### 使用任务模板
1. 点击"📋 使用模板"按钮
2. 选择合适的模板（会议准备、项目启动、购物清单等）
3. 模板会自动创建多个预设任务

### 管理任务
- **完成任务**: 点击任务前的复选框
- **编辑任务**: 点击任务卡片上的编辑按钮（✏️）
- **删除任务**: 点击任务卡片上的删除按钮（🗑️）
- **添加子任务**: 点击任务卡片上的添加按钮（➕）

### 子任务管理
1. 点击任务卡片上的"➕"按钮
2. 输入子任务标题，按回车或点击"添加"
3. 子任务可以独立完成标记
4. 进度条显示完成进度

### 视图切换
- **列表视图**: 传统列表展示，适合查看详细信息
- **看板视图**: 三列看板（待办、进行中、已完成），适合任务流转

在搜索框右侧点击视图切换按钮即可切换。

### 搜索和过滤
- 在搜索框中输入关键词进行实时搜索
- 使用过滤器按分类、优先级、状态筛选任务
- 使用排序选项改变任务显示顺序

### 拖拽排序
在列表视图中，直接拖拽任务卡片可以重新排序。

### 分类管理
1. 点击分类选择框旁边的设置按钮（⚙️）
2. 在弹出的窗口中添加或删除分类
3. 每个分类可以设置名称和颜色

### 数据导入导出
- **导出**: 点击底部的"📤 导出"按钮，保存JSON文件
- **导入**: 点击"📥 导入"按钮，选择之前导出的JSON文件

### 主题切换
点击右上角的主题切换按钮（🌙/☀️）在亮色和暗色主题之间切换。

### 键盘快捷键
- `Ctrl/Cmd + N`: 新建任务（聚焦到任务标题输入框）
- `Ctrl/Cmd + F`: 搜索任务
- `Ctrl/Cmd + /`: 显示快捷键列表
- `Esc`: 关闭弹窗

## 🔧 技术栈

- **前端**: HTML5 + CSS3 + JavaScript (ES6+)
- **数据存储**: LocalStorage
- **PWA**: Service Worker + Web App Manifest
- **通知**: Notification API
- **无需后端**: 纯前端应用

## 💾 数据存储

所有数据都存储在浏览器的 LocalStorage 中，包括：
- 任务列表（包含子任务、重复规则、提醒设置等）
- 分类信息
- 标签数据
- 主题设置
- 任务模板

数据会自动保存，刷新页面后仍然存在。
